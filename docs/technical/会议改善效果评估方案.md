## 📈 二、核心评估指标体系

### 2.1 一级指标 (Primary KPIs)

#### 指标1: 日人均会议数减少率
**定义**: 所有线上会议(包括即时、日程、1v1)的日人均数量变化

**数据来源**: 使用原始数据字段 `日人均线上会议数`

**计算公式**:
```
减少率 = (基线期均值 - 当前期均值) / 基线期均值 × 100%

基线期均值 = (9月日人均线上会议数 + 10月日人均线上会议数) / 2
当前期均值 = 最近4周的平均值
```

**评估标准**:
- 🟢 优秀: 减少 ≥ 20%
- 🟡 良好: 减少 10% ~ 20%
- 🟠 一般: 减少 0% ~ 10%
- 🔴 不达标: 增加或无变化

**意义**: 综合评估所有会议类型的整体负担变化

---

#### 指标2: 日人均会议时长减少率
**定义**: 所有线上会议(包括即时、日程、1v1)的日人均时长变化

**数据来源**: 使用原始数据字段 `日人均线上会议时长(分钟)`

**计算公式**:
```
减少率 = (基线期均值 - 当前期均值) / 基线期均值 × 100%

基线期均值 = (9月日人均线上会议时长 + 10月日人均线上会议时长) / 2
当前期均值 = 最近4周的平均值
```

**评估标准**:
- 🟢 优秀: 减少 ≥ 30分钟/天
- 🟡 良好: 减少 15~30分钟/天
- 🟠 一般: 减少 0~15分钟/天
- 🔴 不达标: 增加或无变化

**意义**: 综合评估所有会议类型的总时长负担变化

---

#### 指标3: 即时会议占比下降
**定义**: 即时会议在所有会议中的占比变化(反映会议计划性提升)

**数据来源**: 使用原始数据字段 `即时会议数` 和 `人的会议数`

**计算公式**:
```
即时会议占比 = 即时会议数 / 人的会议数 × 100%

占比变化 = 基线期占比 - 当前期占比
```

**评估标准**:
- 🟢 优秀: 占比下降 ≥ 10个百分点
- 🟡 良好: 占比下降 5 ~ 10个百分点
- 🟠 一般: 占比下降 0 ~ 5个百分点
- 🔴 不达标: 占比上升

**意义**: 占比下降说明计划性会议增加,临时会议减少,会议管理更规范

---

### 2.2 二级指标 (Secondary KPIs)

#### 指标4: 1v1通话占比
**定义**: 1v1通话在所有会议中的占比(反映沟通方式优化)

**数据来源**: 使用原始数据字段 `1v1通话数` 和 `人的会议数`

**计算公式**:
```
1v1通话占比 = 1v1通话数 / 人的会议数 × 100%

占比变化 = 当前期占比 - 基线期占比
```

**评估标准**:
- 🟢 良好: 占比适度减少 5~15个百分点,说明会议减少
- 🟡 关注: 占比变化 ±5个百分点以内,基本稳定
- 🔴 警示: 占比大幅增加 >15个百分点,可能说明团队协作沟通不畅 ⚠️

**意义**: 适度的1v1占比减少说明沟通方式更灵活高效

---

#### 指标5: 团队会议负担分布均衡度
**定义**: 团队成员会议负担的标准差变化

**计算公式**:
```
均衡度 = STDEV(各成员日人均会议数)

改善率 = (基线期标准差 - 当前期标准差) / 基线期标准差 × 100%
```

**意义**: 标准差降低说明会议负担更均衡,避免个别人过载

---

### 2.3 三级指标 (Monitoring Indicators)

#### 指标6: 周度波动性
**定义**: 各周之间会议数的波动程度

**计算公式**:
```
波动系数 = STDEV(各周均值) / MEAN(各周均值)
```

**解读**: 波动系数下降说明会议节奏更稳定

---

#### 指标7: Top 10 重度会议用户改善率
**定义**: 会议最多的前10人的改善情况

**计算方式**: 单独统计Top 10用户的会议数和时长变化

**意义**: 重点关注高负荷人群的改善效果
