# 会议改善效果评估系统 - 使用指南

## 📚 系统概述

本系统是一个完整的会议改善效果评估解决方案,专为PMO(项目管理办公室)设计,用于评估"固定会议窗口"等改善措施的实施效果。

### 核心能力

✅ **自动数据处理** - 从飞书多维表格同步数据到本地分析
✅ **8项KPI评估** - 全面评估会议改善效果
✅ **趋势分析** - 识别会议数据的长期趋势
✅ **异常检测** - 自动发现异常会议模式
✅ **多维报告** - 管理层、详细、个人、周报4类报告
✅ **可视化** - 自动生成图表和仪表盘

## 🎯 核心指标说明

### 主要KPI (决定成败)

| 指标 | 目标 | 说明 |
|------|------|------|
| 日人均会议数减少率 | ≥15% | 衡量会议数量是否显著减少 |
| 日人均会议时长减少率 | ≥20% | 衡量会议占用时间是否降低 |
| 即时会议占比下降 | ≥10百分点 | 衡量计划性是否提升 |

**达标标准**: 3项全部达标才算改善成功

### 次要KPI (辅助评估)

| 指标 | 目标 | 说明 |
|------|------|------|
| 会议时长效率提升 | ≥10% | 单次会议时长是否缩短 |
| 1v1通话替代率 | ≥5% | 是否用其他方式替代1v1 |
| 团队负担分布均衡度 | 变异系数下降≥10% | 会议负担是否更均衡 |

### 监控指标 (持续关注)

| 指标 | 目标 | 说明 |
|------|------|------|
| 周度波动性 | <0.15 | 会议数是否稳定 |
| Top10重度用户改善率 | ≥20% | 重度会议用户是否改善 |

### 场景3: 个人会议健康检查

**目的**: 为团队成员提供个人会议负担反馈

```python
# 在Python中运行
from meeting_analysis import MeetingDataLoader, MeetingReportGenerator

loader = MeetingDataLoader('../input')
all_data = loader.load_all_data()

# 为张三生成个人报告
user_name = "张三"
user_data = all_data[all_data['user_name'] == user_name]
team_avg = {
    '日人均线上会议数': all_data['日人均线上会议数'].mean(),
    '日人均线上会议时长(分钟)': all_data['日人均线上会议时长(分钟)'].mean()
}

reporter = MeetingReportGenerator('../output')
report = reporter.generate_personal_report(user_data, user_name, team_avg)
print(f"报告已生成: {report}")
```

**频率**: 按需或每月

### 场景4: 异常调查

**目的**: 发现会议数异常的人员或时期

```python
from meeting_analysis import MeetingDataLoader, MeetingDataAnalyzer

loader = MeetingDataLoader('../input')
all_data = loader.load_all_data()

analyzer = MeetingDataAnalyzer(all_data)

# 检测异常
anomalies = analyzer.detect_anomalies('日人均线上会议数', threshold=2.0)
print(f"发现 {len(anomalies)} 个异常数据点:")
print(anomalies)

# 识别Top 10会议最多的用户
top_users = analyzer.identify_top_users('日人均线上会议数', n=10)
print("\nTop 10会议用户:")
print(top_users)
```

**频率**: 按需

## 🔧 高级用法

### 1. 自定义分析周期

```python
from meeting_analysis import MeetingDataLoader, MeetingMetricsCalculator

loader = MeetingDataLoader('../input')
all_data = loader.load_all_data()

# 自定义基线期: 选择8月和9月
baseline = all_data[all_data['month'].isin([8, 9])]

# 自定义当前期: 选择最近2周
current = loader.get_recent_weeks_data(2)

# 计算KPI
calculator = MeetingMetricsCalculator(baseline, current)
kpi_results = calculator.calculate_all_kpis()
```

### 2. 批量生成个人报告

```python
from meeting_analysis import MeetingDataLoader, MeetingReportGenerator

loader = MeetingDataLoader('../input')
all_data = loader.load_all_data()
reporter = MeetingReportGenerator('../output')

# 为所有用户生成报告
team_avg = {
    '日人均线上会议数': all_data['日人均线上会议数'].mean(),
    '日人均线上会议时长(分钟)': all_data['日人均线上会议时长(分钟)'].mean()
}

for user_name in all_data['user_name'].unique():
    user_data = all_data[all_data['user_name'] == user_name]
    reporter.generate_personal_report(user_data, user_name, team_avg)
    print(f"✓ {user_name}")
```

### 3. 相关性分析

```python
from meeting_analysis import MeetingDataLoader, MeetingDataAnalyzer

loader = MeetingDataLoader('../input')
all_data = loader.load_all_data()
analyzer = MeetingDataAnalyzer(all_data)

# 分析会议数和会议时长的相关性
corr = analyzer.analyze_correlation('日人均线上会议数', '日人均线上会议时长(分钟)')
print(f"相关系数: {corr['pearson_correlation']:.4f}")
print(f"p值: {corr['pearson_p_value']:.4f}")
```

## ⚠️ 注意事项

### 数据质量要求

1. **完整性**: 确保基线期(9-10月)和当前期(最近4周)数据完整
2. **一致性**: 所有CSV文件的字段名称必须一致
3. **准确性**: 定期核对飞书数据和本地CSV是否同步

### 最佳实践

1. **定期同步**: 每周五同步飞书数据
2. **定期分析**: 每周一运行分析
3. **归档报告**: 保存每周的分析报告,便于历史对比
4. **分享结果**: 将管理层摘要分享给团队

### 常见错误

1. ❌ **数据不足**: 需要至少2个月度数据(基线期)和4个周度数据(当前期)
2. ❌ **字段缺失**: CSV文件缺少必需字段
3. ❌ **文件命名错误**: 文件名不符合格式要求

## 🆘 故障排查

### 问题1: "数据加载失败"

**原因**: input目录下没有CSV文件或文件格式错误

**解决**:
```bash
# 检查input目录
ls -la input/

# 确认文件格式
head -n 5 input/9月会议详情.csv
```

### 问题2: "未找到基线期数据"

**原因**: 缺少9月或10月的数据文件

**解决**: 确保input目录下有 `9月会议详情.csv` 和 `10月会议详情.csv`

### 问题3: "未找到当前期数据"

**原因**: 缺少周度数据文件

**解决**: 确保至少有4个周度数据文件(格式: `MM.DD-MM.DD会议详情.csv`)

### 问题4: KPI显示"nan"或错误值

**原因**: 数据字段缺失或数据类型错误

**解决**:
```python
# 检查数据
import pandas as pd
df = pd.read_csv('input/9月会议详情.csv')
print(df.columns)  # 查看所有列
print(df.dtypes)   # 查看数据类型
print(df.head())   # 查看前几行
```
