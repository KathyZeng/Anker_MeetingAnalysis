### 6.2 计算说明

- **基线期**: 9月 + 10月的平均值
- **当前期**: 最近4周(10.20-11.16)的平均值
- **变化率**: (当前期 - 基线期) / 基线期 × 100%
- **变异系数(CV)**: 标准差 / 均值,用于衡量离散程度
- **Z-score**: (值 - 均值) / 标准差,用于异常检测
