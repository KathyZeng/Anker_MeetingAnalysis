# 🚀 快速更新指南

## 三步更新仪表盘

```bash
# 1️⃣ 放入新数据
cp 11月会议详情.csv input/

# 2️⃣ 生成仪表盘
python3 generate_full_dashboard.py

# 3️⃣ 查看结果
open output/meeting_dashboard_full.html
```

---

## 📋 CSV文件必需列

| 列名 | 示例 |
|------|------|
| user_name | 张三 |
| 日人均线上会议数 | 3.5 |
| 日人均线上会议时长(分钟) | 65.3 |
| 即时会议 | 12 |
| 日程会议 | 18 |
| 1v1通话数 | 8 |

---

## 💡 命名建议

### 月度数据
```
9月会议详情.csv
10月会议详情.csv
11月会议详情.csv
```

### 周度数据
```
10.20-10.26会议详情.csv
10.27-11.2会议详情.csv
11.03-11.09会议详情.csv
```

## ⚠️ 注意事项

- ✅ 使用UTF-8编码
- ✅ 确保列名正确
- ✅ 数字列为纯数字
- ❌ 避免特殊字符
- ❌ 避免文件名中的空格
